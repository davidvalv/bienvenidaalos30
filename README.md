# Funeral por la veintena de Candela

Página estática preparada para GitHub Pages.

## Estructura

- `index.html`: contenido.
- `styles.css`: estilos responsive y animaciones.
- `script.js`: revelado en scroll, contadores y confeti.
- `assets/`: imágenes opcionales.

## Assets incluidos

Las imágenes están copiadas en `assets/` y la página apunta directamente a sus nombres reales.
Si alguna falta, se muestra un fallback gráfico para no romper el diseño.

## Publicación en GitHub Pages

Sube estos archivos a la rama que uses para Pages. Cuando tengas el dominio `.es`, crea un archivo
`CNAME` con el dominio exacto y configura los DNS en GoDaddy apuntando a GitHub Pages.
